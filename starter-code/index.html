Instructions

Concentration, also known as Match Match, Memory. Pelmanism, Shinkei-suijaku, Pexeso, or Pairs, is a card game in which all of the cads are laid facedown on a surface and two cards are flipped up over each turn. The object of the game is to turn over pairs of matching cards.

About

Memory Game is fun to play on your own, with a group of friends or family. Memory Game is challenging and can become very competitive when playing in a group. Memory Game exercises the mind and this be beneficial to your daily activities.
